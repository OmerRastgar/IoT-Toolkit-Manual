# Additional Modules Overview

Extended modules for enhanced IoT Toolkit functionality.

## Module Types

| Module | Purpose | Interface |
|--------|---------|-----------|
| [I2C Multiplexer](i2c-multiplexer.md) | Resolve I2C address conflicts | I2C |
| [Voice Module](voice.md) | Audio output and alerts | I2C/UART |
| [Camera Module](camera.md) | Image capture and monitoring | Specialized |

## Use Cases

### I2C Multiplexer
- Connect multiple sensors with same I2C address
- Expand available I2C channels
- Isolate faulty devices

### Voice Module
- Audio alerts for threshold violations
- Voice announcements
- Status feedback

### Camera Module
- Visual monitoring
- Image capture on events
- Remote surveillance

## Quick Start

1. [I2C Multiplexer](i2c-multiplexer.md) - If you have address conflicts
2. [Voice Module](voice.md) - For audio feedback
3. [Camera Module](camera.md) - For visual monitoring

!!! tip "Optional Modules"
    These modules are optional but add significant functionality to your IoT Toolkit.
